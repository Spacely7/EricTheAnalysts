Cement Sales Dashboard

Overview: The Cement Demand Forecasting Dashboard is a data analytics project focused on understanding and predicting trends in cement demand. Using historical data, it applies machine learning models to forecast future demand and visualize key metrics like average demand, peak production, and sales. Built entirely with Streamlit, the dashboard offers an intuitive interface with interactive charts and summaries that make complex data easy to understand. This project demonstrates my skills in data analysis, machine learning, and dashboard development.

Description: This project is an interactive dashboard built with Streamlit that uses machine learning to forecast cement demand. It analyzes historical data to provide insights on average demand, peak months, production, and sales. The dashboard features clear visualizations and a forecast chart that helps users understand trends and predict future demand. It's a valuable tool for making data-driven decisions in cement production and supply chain planning.

Tools: Pycharm, Excel, Python
