About This Project

This project demonstrates real-time object detection using TensorFlow.js and the COCO-SSD model. It runs entirely in the browser, without any backend server, making it lightweight and easy to deploy.

Technologies used: JavaScript, Bootstrap 5, TensorFlow.js, COCO-SSD.
